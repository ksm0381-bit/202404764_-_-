■ Google 로그인 활성화 방법 (Firebase 설정)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. console.firebase.google.com → 새 프로젝트 생성
2. Authentication → Google 로그인 활성화
3. Railway URL을 승인된 도메인에 추가
4. 프로젝트 설정 → 앱 추가(웹) → Firebase config 복사
5. 대시보드 → ⚙️ 설정 탭 → Firebase 설정에 붙여넣기 → 저장
