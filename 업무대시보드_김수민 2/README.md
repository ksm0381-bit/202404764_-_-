## 📊 프로젝트 소개

NOVA Creative(마케팅·기획사) 업무 환경을 가상으로 구성한 **단일 파일 HTML 종합 대시보드**입니다.  
별도 서버나 라이브러리 설치 없이 브라우저에서 바로 실행됩니다.

🔗 **GitHub Pages 라이브 데모**: [https://kimsoomin.github.io/nova-dashboard](https://kimsoomin.github.io/nova-dashboard)

---

## 🖥️ 주요 기능

| 섹션 | 내용 |
|------|------|
| **KPI 카드** | 진행 프로젝트 수 · D-1 마감 · 주간 완료 · 팀원 온라인 등 6개 지표 |
| **AI 데일리 브리핑** | 오늘의 Top 3 우선 업무 AI 요약 |
| **할 일 (To-Do)** | 탭 필터(전체/긴급/회의/보고/일반) · 클릭 완료 체크 · 항목 추가 |
| **결재·문서 트래커** | 결재대기 / 검토중 / 반려 / 완료 현황 |
| **프로젝트 마일스톤** | 5개 프로젝트 진행률 바 |
| **실시간 캠페인 현황** | 노출수 · CTR · 전환 · 팔로워 메트릭 |
| **회의실 현황** | 시간대별 예약 현황 시각화 |
| **팀원 현황** | 온라인 상태 · 업무 부하도 |
| **미니 캘린더** | 5월 일정 하이라이트 |
| **외부 출장·미팅** | 오늘 출장/미팅 현황 |
| **활동 피드** | 최근 팀 활동 로그 |
| **차트** | Chart.js 막대 · 도넛 차트 |

### 🖱️ 클릭 인터랙션
모든 주요 요소를 클릭하면 **상세 정보 모달**이 팝업됩니다.
- KPI 카드 6개, 프로젝트 마일스톤, 팀원 행, 결재 문서
- AI 브리핑 항목, 실시간 메트릭 셀, 회의실 행
- 캘린더 이벤트, 활동 피드, 공지 항목, 출장 행

---

## 🛠️ 기술 스택

- **HTML5 / CSS3 / Vanilla JS** — 프레임워크 없음
- **Chart.js 4.4.1** — CDN 로드 (막대·도넛 차트)
- **다크 모드 디자인** — CSS 변수 기반 테마

## 🚀 실행 방법

```bash
# 1. 레포지토리 클론
git clone https://github.com/kimsoomin/nova-dashboard.git
cd nova-dashboard

# 2. index.html을 브라우저로 열기
open index.html   # macOS
start index.html  # Windows
```

또는 `index.html` 파일을 브라우저로 드래그 앤 드롭하면 바로 실행됩니다.
